2. 문제설명

정수 N이 주어진다.

1부터 N까지의 모든 숫자 중에서 아래 조건을 만족하는 숫자를 오름차순으로 출력하라.

[조건]

어떤 숫자 X가 있을 때,

X의 각 자리 숫자의 합을 구한다.
X를 각 자리 숫자로 나누었을 때, 모두 나누어 떨어져야 한다.

즉,

자리수 합 = S
각 자리 숫자 d에 대해
→ X % d == 0 이어야 한다 (단, d ≠ 0)

이 두 조건을 모두 만족하는 숫자만 출력한다.

[주의사항]
숫자에 0이 포함된 경우는 무조건 제외 (0으로 나눌 수 없음)
자리수는 직접 분리해서 처리해야 한다 (문자열 사용 금지)
함수로 분리해서 구현할 것
[예시 설명]

예를 들어 X = 12 라면

자리수: 1, 2
자리수 합: 3
12 % 1 == 0 ✔
12 % 2 == 0 ✔

→ 조건 만족 → 출력 대상

2. 문제설명

정수 N이 주어진다. (1 ≤ N ≤ 1000)

길이가 N인 정수 배열이 입력된다.

이때, 배열에서 연속된 구간 [i ~ j] 중 아래 조건을 만족하는 구간의 개수를 구하라.

[조건]

구간의 모든 원소에 대해:

구간의 합 = 구간 길이 × 어떤 정수 K
즉, 평균이 정수여야 한다
그 구간에서
최댓값 - 최솟값 ≤ K
[설명 정리]

구간 [i ~ j]에 대해:

sum = a[i] + ... + a[j]
len = j - i + 1

→ sum % len == 0 이어야 하고
→ K = sum / len

그리고
→ (max - min) ≤ K 이어야 한다

[주의사항]
모든 구간을 다 보면 시간 초과 위험 있음 → 흐름 설계 중요
매번 max/min 새로 구하면 비효율 → 어떻게 처리할지 고민
함수로 분리할 것
배열은 포인터로 전달할 것
